This section is obsolete and remains for backward compatibility. it is currently hidden and can be ignored The Test Plan folder path in the project where the scripts will be loaded.
Available paths can be found in the application interface: Test Management -> Test Plan
The path must begin with "Subject" and be separated with backslashes (\).
The path is case-sensitive!
Important: For Performance Center Server below version 12.60:
  • the specified folder must exist at the time of the build (the plugin can not create folders independently).
  • Scripts saved in Git subfolders will be synchronized to the specified subject and in case of duplicated script names in Git, the last uploaded script to the project will overwrite the previously uploaded script(s) with the same name.
